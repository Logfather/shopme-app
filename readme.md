# ShopMe

ShopMe ist eine moderne Android Einkaufslisten-App mit Multi-Store Unterstützung, Realtime Synchronisation und Offline-First Architektur.

Die App wurde mit **Kotlin, Jetpack Compose und Firebase Firestore** entwickelt und folgt einer **Clean Architecture**.

---

# Features

- Einkaufslisten für mehrere Supermärkte
- Realtime Synchronisation über Firestore
- Offline-First Vorbereitung mit Room
- Sprachsteuerung für schnelle Eingabe
- Multi-Device Synchronisation
- Optimistic UI Updates
- Konfliktauflösung bei parallelen Änderungen
- moderne Compose UI

---

# Technologie-Stack

| Bereich | Technologie |
|------|------|
| Sprache | Kotlin |
| UI | Jetpack Compose |
| Backend | Firebase Firestore |
| Lokaler Cache | Room |
| Concurrency | Kotlin Coroutines + Flow |
| Architektur | Clean Architecture |
| State Management | State Machine + Reducer Pattern |

---

# Architektur

Die App folgt einer **Clean Architecture mit klar getrennten Schichten**.
